Module nerve_lib.manage_access
==============================
Manage Access releated function on MS.

Example:
-------
    >>> from nerve_lib import MSHandle
    >>> from nerve_lib import MSUser
    >>> with MSHandle("testms.nerve.cloud") as ms_handle:
    >>>     user = MSUser(ms_handle)
    >>>     user.get()
    <current user-list>
